>>> ABOUT THE PROJECT <<<

Greetings, wanderer! You have reached a page served by my very own web server, written in the C language with my bare hands, without any of those fancy Apaches and other newfangled things!

Yes indeed, this very page was just delivered to you over the HTTP/1.0 protocol by a tiny server only a couple hundred lines of code long!

>>> WHAT THE SERVER CAN DO <<<

  • Listens on port 7980
  • Accepts connections via BSD sockets
  • Understands GET and HEAD requests
  • Serves files from the webroot directory
  • Honestly says 404 when a file is not there
  • Written in pure C with no external libraries

>>> HOW IT WORKS <<<

  1. The server creates a socket and binds it to a port
  2. Waits for visitors with listen and accept
  3. Reads the first line of the request byte by byte
  4. Opens the requested file and sends it straight to your browser!

>>> FUTURE PLANS <<<

  • MIME type support (for now everything is sent as-is!)
  • Visitor logs written to a file
  • Maybe even HTTP/1.1... but no promises
